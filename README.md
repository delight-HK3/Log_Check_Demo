# Log Check Demo
Dozzle 을 활용한 로그 분석용 더미 프로젝트

<br>

## 아키텍처
<img width="1265" height="586" alt="화면 캡처 2026-01-16 103803" src="https://github.com/user-attachments/assets/ef19fd27-978a-4f4a-ace6-fdd07f9959ec" />

현재 계획은 Dozzle을 커스텀하여 단순 로그 Viewer에서 로그 일간별 백업, 기간별 조회 이렇게 기능추가할 예정

- 먼저 기간별 로그 검색 기능 개발
    - 해당 기능은 Vue를 수정하여 프론트단에서 랜더링하면 되는걸로 현재로써 예상
    - 기존에 진행되는 로그 컴포넌트 위에 기간단위로 조회한 로그를 위에 덧 씌운형태로 보이게 하면 되는걸로 예상한다.
- 일간 로그저장
    - 해당 기능은 실질적으로 저장경로를 지정하고 UI상으로 로그를 다운로드가 가능하도록 설계
